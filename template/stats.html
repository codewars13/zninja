UNHCR Statistics

UNHCR Malta during refugee crisis need better solution to show and update their statistical asylum trends. Personel inside UNHCR works mostly with Excel sheets to present data, so our main focus was to transform that excel data into something user friendly and acceptable on the web. As a frontend developer, I implemented Google Charts and with backend developer we set rules to populate Excel sheet which resulted simple to use "upload excel file only" application.
